## nan 커뮤니티 게시글 목록 조회

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Community
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 하

### 설명
- 사용자는 커뮤니티 게시판에서 자유게시판 글들의 목록을 볼수있다.
- -정렬 기준
- •  최신순
- •  오래된순
- •  조회수 많은순
- •  댓글 많은순
- 조회 가능 항목
- • 제목
- • 작성자 이름
- • 조회수
- • 댓글 갯수
- • create at (작성된 시각)
- • update at (마지막으로 수정된 시간)

### 비고
- 게시글이 수정되었다면 목록의 제목에서 (수정됨) 이라는 표시가 보입니다.
- 수정된 일시는 목록에 마우스 hover 하면 보입니다.

## nan 게시글 댓글 작성하기(커뮤니티 기능)

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Community
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 하

### 설명
- 사용자는 특정 게시글의 댓글을 작성할 수 있다.
- • 작성 항목 : 내용
- - 부모 댓글(parent_id)을 지정하여 대댓글 또는 대대댓글을 작성할 수 있습니다.
- 최대 3단계(댓글→대댓글→대대댓글) 까지만 등록 가능합니다.

### 비고
- 로그인 필요
- 비로그인 시, 로그인 페이지로 유도
- 글자 제한수 ( 2~500까지)
- - 4단계 이상의 depth 요청 시 안내문구 반환
- → 추후 결정

## nan 게시글 댓글 목록 조회하기

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Community
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 하

### 설명
- 사용자는 특정 게시글의 댓글 목록(부모 댓글 기준)을 조회할 수있습니다.
- - 댓글에는 작성자와 작성일이 표시됩니다.
- • 조회 조건:
- •  부모 댓글(Depth 1)은 기본적으로 전체 표시됩니다.
- •  사용자는 “더보기” 버튼을 클릭하여
- 해당 부모 댓글의 하위 댓글(대댓글,대대댓글)을 불러올 수 있습니다.
- • 각 댓글에는 작성자 닉네임,내용,작성일,작성시간,프로필 이미지 가 보입니다.

### 비고
- (특이사항 없음)
