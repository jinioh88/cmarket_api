## nan 사용자 차단 기능

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Report
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 중

### 설명
- 유저 프로필 페이지에서 사용자는 특정 사용자를 차단할 수 있습니다.
- • 차단 기능 수행 시
- - 로그인 유저가 차단한 유저와는 채팅할수 없다.
- ◦ 차단된 사용자의 게시글과 댓글을 볼 수 없음
- ◦ 차단된 사용자 또한 차단한 사용자의 게시글과 댓글을 볼 수 없음
- ◦ 서로의 콘텐츠(게시글/댓글)에 대한 상호작용 차단

### 비고
- 차단하기 버튼 클릭으로 실행
- 로그인 필요

## nan 유저 신고하기

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Report
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 중

### 설명
- 로그인을 한 유저는 유저 프로필 페이지 내에서 다른 사용자가 부적절한 행위를 한 경우 해당 사용자를 신고할 수 있습니다.
- 신고 시 사용자는 신고 항목을 선택하거나 직접 입력할 수 있습니다.
- •신고 항목 예시:
- • 욕설,비방,괴롭힘
- • 사기, 허위 거래 시도
- • 음란물 또는 불건전 행위
- • 스팸/광고성 메시지
- • 만 14세 미만 유저입니다.
- • 사용자 닉네임 신고
- • 사용자 프로필 이미지 신고
- • 기타(직접 입력)
- •  신고 상세 사유 최대 300자

### 비고
- • 로그인 필요
- •한 사용자당 동일 유저 1회만 신고가능
- • 다중 선택 가능하도록 만들기
- • 스크린 샷과 같은 이미지 첨부 가능
- (최대 3장)
- • 관리자는 신고 내역을 검토하고 사용자에 대한 조치를 취할 수 있어야 합니다.
- 신고 항목을 선택해야 합니다.
- • 이미 신고 접수된 유저에 대해
- 중복 신고 요청시 “이미 신고된 유저입니다.” 라는 안내 문구를 띄워야 합니다.

## nan 상품 신고하기

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Report
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 중

### 설명
- • 사용자는 부적절하거나 불법적인 상품을 발견했을 경우 상품 상세페이지 내에서 신고할 수 있습니다.
- • 신고 시 사용자는 신고 항목을 선택하거나 직접 입력할 수 있습니다.
- • 신고 항목 예시 :
- • 허위/사기성 상품
- • 불법 또는 금지 품목
- • 부적절한 이미지
- • 중복 게시물
- • 스팸/광고성 게시물
- • 대리 결제/구매/판매 행위
- • 전문 판매 업자
- • 기타(직접입력)
- •  신고 상세 사유 최대 300자

### 비고
- • 로그인 필요
- • 로그인하지 않은 유저가 신고 시도시 로그인 페이지로 유도
- • 한 상품당 한 사용자 1회만 신고 가능
- • 신고시 스크린 샷과 같은 이미지 첨부 가능
- → (최대 3장)
- • 이미 신고 접수된 상품에 대해 중복 신고 요청시
- “이미 신고된 상품입니다.” 라는 안내 문구를 띄워야 합니다.
- •관리자는 신고 내역을 조회 및 처리가능

## nan 커뮤니티 게시글 신고하기

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Report
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 하

### 설명
- 사용자는 커뮤니티 내에서 부적절한 게시글을 발견할 경우 게시글 상세페이지 내에서 신고할 수 있습니다.
- 신고시 사용자는 신고 항목을 선택하거나 직접 작성할 수 있습니다.
- •신고 항목 예시 :
- • 욕설/비방/혐오 표현
- • 도배 게시물
- • 음란물/불건전 콘텐츠
- • 스팸/광고성 게시물
- • 자해 또는 자살 의도를 포함
- • 기타(직접 입력)

### 비고
- • 로그인 필요
- • 로그인하지 않은 유저가 신고 시도시 로그인 페이지로 유도
- • 관리자는 신고 내역을 검토 후 게시글 조치 가능
- • 한 게시글당 한 사용자 1회만 신고 가능
- • 신고시 스크린 샷과 같은 이미지 첨부 가능
- → (최대 3장)
- • 이미 신고 접수된 게시글에 대해 중복 신고 요청시
- “이미 신고된 게시글입니다.” 라는 안내 문구를 띄워야 합니다.
