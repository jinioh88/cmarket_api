## nan 회원 목록 조회

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Admin
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 관리자 권한을 가진 유저는
- 어드민 페이지의 회원 관리 메뉴에 접속해 회원 목록을 조회할 수 있습니다.
- 조회된 항목에 대해서 페이지네이션, 필터링, 검색 기능을 제공합니다.
- - 검색 기능
- 검색 필드:
- • 닉네임
- • 이메일
- • 이름
- • ID(PK)
- 목록에서 제공 가능한 항목
- •  회원 ID (PK)
- •  닉네임
- •  이름
- •  이메일
- •  생년월일
- •  상태
- •  권한
- •  회원가입일(Create At)
- •  지역(시,구)

### 비고
- 페이지네이션
- → 10개씩
- 필터링
- → 상태(활성/비활성/탈퇴요청) 과 권한(관리자/일반회원)
- • 목록 조회시 정렬 (PK값으로 오름차순)

## nan 어드민 상품 목록 조회 및 검색

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Admin
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 관리자 권한을 가진 사용자는 어드민 페이지의 게시글 관리 메뉴에서 상품 목록 및 판매요청 게시글을 조회하거나 검색할 수 있습니다.
- •검색 기능을 통해 상품 제목 또는 작성자 닉네임을 입력하면,
- 해당 단어가 포함된 상품을 정규화된 문자열 기준(대소문자 구분 없이) 으로 조회할 수 있습니다.
- • 다중 키워드 검색 시, 모든 단어를 포함한 상품만 조회됩니다.검색 결과는 기본적으로 상품 제목 → 작성자 이름 순으로 정렬됩니다.
- • 조회 / 검색 시 표시 항목
- • 상품 ID (PK)
- • 대표 이미지 (썸네일)
- • 판매 상품 제목
- • 작성자 닉네임
- • 거래 상태(거래중,예약중,거래완료)
- • 작성일자 (created_at)
- • 수정일자 (updated_at)
- • 조회수
- • 찜 갯수

### 비고
- 관리자 권한 필요
- 추후 기능 확장 시,
- • 판매 상태별 필터링(판매중 / 예약중 / 거래완료)
- • 카테고리별 필터링
- • 신고 상품 우선 표시
- 등의 기능 추가 고려
