## FR-015 상품 통합검색

<a id="fr-015"></a>

- **요구사항 ID:** `FR-015`
- **대분류:** Search
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 하나의 검색창에 키워드를 입력하여
- 관련 상품을 검색할 수 있다.
- 검색어는 상품 제목, 상품 설명, 카테고리명 에부분적으로 포함되어있으면 결과에 표시된다.
- -검색 결과는
- 제목 일치 > 설명 일치 > 카테고리 일치 순으로 우선 정렬되며,
- 다중 단어 검색시 모든 단어가 포함된 상품만 조회된다

### 비고
- 전체 상품 대상 통합 검색 기능 제공
- 로그인 여부와 관계없이 이용 가능
- • 검색 기준 : 부분 포함(대소문자 구분없음)
- • 검색 필드 :
- 상품제목(product.name),설명(description),
- 카테고리명(category.name)
- • 우선 순위 : 제목 > 설명 > 카테고리
- • 다중 키워드 : 한글 정규화 AND 조건으로 처리
- • 불필요한 특수문자 및 이모지는 검색시 무시 처리한다.
- -향후 확장 기능 : 자동완성 , 인기 검색어 ,
- 정렬 기준(최신순/인기순)해시태그 검색기능

## FR-016 카테고리 필터링

<a id="fr-016"></a>

- **요구사항 ID:** `FR-016`
- **대분류:** Search
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 검색 결과 또는 전체 상품 목록에서
- 필터 조건을 선택하여 원하는 상품만 조회할 수 있다.
- - 필터는 반려동물 종류, 상품 카테고리 ,가격대,지역으로 구성됩니다.
- 각 필터는 독립적으로 또는 조합하여 적용할 수 있습니다.
- 사용자가 특정 필터를 선택하면,
- 해당 조건에 맞는 상품만 화면에 즉시 반영되며,
- 선택된 필터는 한눈에 확인할 수 있습니다.
- - 동물 종류 항목:
- 강아지,고양이,토끼,햄스터,기니피그,페럿,친칠라,고슴도치,
- 잉꼬,앵무새,가나리아,모란앵무,
- 도마뱀,뱀,거북이,게코,
- 금붕어,열대어,체리새우,
- 사마귀,딱정벌레,거미

### 비고
- - 필터 구성 항목 :
- 1 반려동물 대분류 : 전체/포유류/조류/파충류/수생동물/곤충,절지동물)
- 2 상품 카테고리 : 사료/간식, 장나감,하우스,의류/액세서리,
- 이동용품,훈련용품
- 3 상품 상태 : 새 상품/거의 새것/사용감 있음/수리 필요
- 4가격대:1만원 이하/1만원 이상/5만원 이상/10만원 이상
- 5 지역 : 시,도 및 시,군,구 단위 선택
- 서울특별시 / 경기광역시 / 인천광역시 / 부산직할시 / 대구광역시 / 광주광역시 / 대전광역시 / 울산광역시
- -필터 변경 시 페이지 새로고침 없이 실시간 결과 반영(프론트 처리 기준)

## FR-028 정렬 기능

<a id="fr-028"></a>

- **요구사항 ID:** `FR-028`
- **대분류:** Search
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 상품 목록을 다양한 기준으로 정렬하여 조회할 수 있습니다.
- • 정렬 기준
- ◦ 최신순 (기본 정렬)
- ◦ 낮은 가격순
- ◦ 높은 가격순
- ◦ 찜 많은 순
- • 각 기준별로 오름차순 / 내림차순 선택 가능

### 비고
- 쿼리 파라미터로 정렬 기준 전달이 필요하다.
