## FR-029 알림 기능

<a id="fr-029"></a>

- **요구사항 ID:** `FR-029`
- **대분류:** Notification
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 하

### 설명
- 사용자는 알림 기능을 통해 새로운 소식을 실시간으로 확인할 수 있습니다.
- • 채팅 관련 알림
- ◦ 새로운 채팅이 생성되었을 때
- ◦ 새로운 메시지가 도착했을 때
- • 거래 관련 알림
- ◦ 찜한 상품 거래 상태 변경 시 알림
- • 어드민 재제 알림
- • 내 게시글이 삭제 당했을때 알림
- • (판매요청 게시판)내가 작성한 댓글에, 댓글이 달렸을 경우
- - (판매요청 게시판)내가 작성한 게시글에 댓글이 달렸을 경우

### 비고
- 푸시 알림 또는 알림함 저장 방식 지원
- 로그인 필요
- 추후 고려 기능
- → 내가 찜한 상품의 가격이 변동되었을때
