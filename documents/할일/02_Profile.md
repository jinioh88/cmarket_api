## FR-005 마이페이지 조회

<a id="fr-005"></a>

- **요구사항 ID:** `FR-005`
- **대분류:** Profile
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 마이페이지를 통해 자신의 기본 정보를 조회할 수 있습니다.
- • 조회 가능한 항목
- ◦ 프로필 이미지
- ◦ 닉네임
- ◦ 이름
- ◦ 소개글
- ◦ 내가 찜한 상품 목록
- ◦ 내가 등록한 상품 목록
- ◦ 내가 등록한 판매 요청 목록
- ◦ 내가 차단한 유저 목록
- ◦ 생년월일
- ◦ 이메일
- ◦ 거주지(시.구)
- ◦ 가입일

### 비고
- 로그인 필요

## nan 비밀번호 변경

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Profile
- **HTTP:** `PATCH`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 비밀번호를 변경할수있다
- 1. 변경할 새 비밀번호와, 새 비밀번호 확인 칸에 비밀번호를 입력해야 합니다.
- 2. 비밀번호 변경 버튼을 누르면 비밀번호가 변경됩니다.

### 비고
- 비밀번호 유효성(특수기호, 대문자, 소문자, 숫자 포함) 검사 필요
- →  유효성 검증 실패시 “비밀번호는 대소문자 숫자 특수기호 조합이어야 합니다.” 라는 안내 문구를 띄워야만 합니다.
- 새 비밀번호 창과 새 비밀번호 확인이 서로 일치하는지 검증 필요
- → 불일치할 경우 “비밀번호가 일치하지 않습니다.” 라는 안내 문구를 띄워야만 합니다.

## nan 차단한 유저 목록 조회

<a id="nan"></a>

- **대분류:** Profile
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 마이페이지에서 자신이 차단한 모든 유저의 목록을 조회할 수 있다. 목록은 최신 차단 순서로 정렬된다.
- • 조회 되는 항목:

- • 프로필 이미지

## nan 유저 차단 해제

<a id="nan"></a>

- **요구사항 ID:** `nan`
- **대분류:** Profile
- **HTTP:** `DELETE`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 차단한 유저 목록에서 특정 유저의 차단을 해제할 수 있다.
- 차단 해제 시 해당 유저와 다시 상호작용(채팅,댓글 보기 등)이 가능해진다.
- 차단 해제가 완료되면 차단 목록에서 즉시 제거된다

### 비고
- 로그인 필요
- 차단 해제가 완료되면 토스트를 띄움
