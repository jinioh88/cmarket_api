## FR-008 판매 상품 등록

<a id="fr-008"></a>

- **요구사항 ID:** `FR-008`
- **대분류:** Product
- **HTTP:** `POST`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 상품 기본정보 입력
- • 동물 종류 (예: 파충류, 포유류 등)
- • 반려동물 상세 종류 선택
- • 상품 카테고리
- • 상품명 입력 (최대 50자)
- • 상품 설명 입력 (최대 1000자)
- • 판매 가격 (숫자만 입력가능하도록)
- • 상품 상태 (새 상품, 사용감 있음 등)거래 방식 정보 입력
- • 이미지 (최대 5장이며 대표이미지는 1장으로 한다.)
- • 지역 (시/구 선택해야하며 필수로 적어야 한다.)
- • 택배 거래
- • 선호하는 만남 장소

### 비고
- 상품명: 최소 2자 ~ 최대 50자
- 상품 설명: 최대 1000자
- 상품 이미지는 선택 업로드 항목
- - 이미지 용량 제한 (이미지 한장당 5mb, 총 25mb)
- 로그인 필요

## FR-012 판매 상품 수정

<a id="fr-012"></a>

- **요구사항 ID:** `FR-012`
- **대분류:** Product
- **HTTP:** `PATCH`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 자신이 등록한 상품 정보를 마이페이지에서 수정할 수 있습니다.
- • 수정 가능한 항목
- ◦ 반려동물 (대분류)
- ◦ 반려동물 상세 종류 선택
- ◦ 상품 카테고리
- ◦ 상품명 (최대 50자)
- ◦ 상품 설명 (최대 작성 가능 길이 1000자)
- ◦ 가격
- ◦ 상품 상태 ( 새 상품,거의 새것,사용감 있음,
- 수리필요)
- ◦ 상품 이미지(최대 등록 가능 이미지 갯수 5개)
- ◦ 택배거래 가능 여부
- ◦ 거래 희망 지역(시.구)
- ◦ 선호하는 만남 장소

### 비고
- 본인이 등록한 상품만 수정 가능
- 로그인 필요
- 상품 이미지 갯수
- (대표 이미지 1장,서브 이미지 4장)
- - 판매 상품 수정인지, 판매요청 수정인지 보여야 합니다.

## FR-013 거래 상태 변경(판매 상품/ 판매요청)통합

<a id="fr-013"></a>

- **요구사항 ID:** `FR-013`
- **대분류:** Product
- **HTTP:** `PATCH`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자는 마이페이지에서 본인이 등록한 판매 상품 또는
- 판매 요청 게시글의 거래 상태를 변경할 수 있습니다.
- 거래 상태는 게시글의 종류에 따라 다르게 표시됩니다.
- “판매중”→”예약중”→”거래완료”등으로 변경할수있다
- 삽니다 → 예약중 → 거래완료거래 상태 변경 시
- 별도의 저장 버튼 없이 즉시 적용됩니다.

### 비고
- 본인 상품만 수정 가능/ 로그인 필요
- 적용 대상:
- • 판매 상품
- • 판매 요청 게시글
- 마이페이지 들어가서 있는
- 거래 상태 버튼을 눌러서 변경하는 것

## FR-031 상품 조회수

<a id="fr-031"></a>

- **요구사항 ID:** `FR-031`
- **대분류:** Product
- **HTTP:** `GET`
- **구분:** 기능
- **우선순위:** 상

### 설명
- 사용자가 상품 상세 페이지에 접속 할 떄마다 상품의 조회수가 1씩 증가한다.

- Product 테이블의 view_count 컬럼으로 저장하여
- 캐시 또는 DB업데이트 방식으로 성능 고려후 결정
- - 본인이 등록한 상품 상세페이지 접속해도 count 증가
