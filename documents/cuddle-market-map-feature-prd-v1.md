# 📄 커들마켓 지도 기반 생활 서비스 기능 기획서 (v1.0)

---

## 1. 기능 개요

### 기능명

반려동물 생활 지도 서비스

### 목적

- 사용자의 **앱 체류시간 증가**
- 커들마켓을 **거래 플랫폼 → 생활 플랫폼**으로 확장
- **반려동물 기반 위치 서비스 차별화**
- 향후 **광고/추천 기반 수익 모델 확보**

### 데이터 소스 & 기술 스택

- **지도 SDK**: Naver Maps JavaScript API v3
- **장소 데이터**: 네이버 지도 API (검색/장소 API)
- **"반려동물 동반 가능" 데이터**: 네이버 API에서 직접 제공하지 않으므로 별도 확보 전략 필요
  - 1안: 공공데이터포털 / 외부 오픈 데이터 활용
  - 2안: 관리자 직접 등록 및 관리
  - 3안: 사용자 제보 기반 크라우드소싱 (2차 이후)

---

## 2. 타겟 사용자

- 반려동물을 키우는 보호자
- 특히 아래 니즈가 있는 사용자
  - 근처 동물병원을 찾고 싶은 사용자
  - 반려동물과 함께 갈 수 있는 장소를 찾는 사용자
  - 응급 상황에서 빠르게 병원을 찾고 싶은 사용자

---

## 3. MVP 기능 범위 (1차 출시)

### 3-1. 공통 기능 (핵심)

- 지도 기반 UI 제공
- 사용자 위치 기반 주변 장소 표시
- 카테고리별 필터 제공
- 지도 내 버튼을 통한 빠른 필터링 (네이버 지도 스타일)
- 장소 클릭 시 상세 정보 표시

---

### 3-2. 카테고리별 기능

#### 🏥 동물병원 찾기

- 사용자 주변 동물병원 표시
- 필터 기능
  - 24시간 운영
  - 응급 진료 가능
  - 특수동물 진료 가능
    - 파충류
    - 조류
- 추천 병원 표시 (광고 가능 영역)

---

#### ☕ 카페 찾기

- 반려동물 동반 가능 카페만 표시 (동반 불가 장소는 미노출)
- 추천 카페 표시 (광고 가능)

---

#### 🍽 식당 찾기

- 반려동물 동반 가능 식당만 표시 (동반 불가 장소는 미노출)
- 추천 식당 표시 (광고 가능)

---

#### 🏨 숙소 찾기

- 반려동물 동반 가능 숙소만 표시 (동반 불가 장소는 미노출)
- 추천 숙소 표시 (광고 가능)

---

### 3-3. 리뷰 기능

- 각 장소(병원/카페/식당/숙소)에 대해
  - 리뷰 조회 가능
  - 리뷰 작성 가능

### 3-4. 상세 정보 화면

- 장소 마커 클릭 시 표시할 정보:
  - 장소명
  - 주소
  - 전화번호 (탭하여 바로 전화 가능)
  - 영업시간
  - 카테고리 태그
  - 리뷰 목록 (조회)
  - 리뷰 작성 버튼
  - 추천 여부 표시
- 외부 지도앱 연결 버튼 (네이버 지도앱으로 길찾기 대체)

### 3-5. 위치 권한 처리

- 위치 권한 허용 시: 현재 위치 기반 지도 표시
- 위치 권한 거부 시: 기본 위치(서울 시청, 위도: 37.5666, 경도: 126.9784)로 진입

---

## 4. 필터 시스템 (핵심 기능)

- 지도 화면 내에서 바로 접근 가능해야 함
- 버튼 형태 UI (네이버 지도 스타일)

### 필터 유형

#### 공통 필터

- 카테고리 (병원 / 카페 / 식당 / 숙소)

#### 병원 전용 필터

- 24시간 운영
- 응급 진료
- 특수동물 진료 (복수 선택 가능)

#### 장소(카페/식당/숙소)

- 별도 필터 없음 (반려동물 동반 가능 장소만 노출되므로 필터 불필요)

---

## 5. 추천(광고) 영역

- 각 카테고리별로 “추천” 데이터 존재
- 일반 데이터와 구분되어 표시
- 추후 광고 상품으로 확장 가능

---

## 6. 제외 기능 (MVP 범위 외)

- 예약 기능 ❌
- 결제 기능 ❌
- 길찾기 기능 ❌ (외부 지도앱 연결로 대체)

---

## 7. 핵심 사용자 시나리오

### 시나리오 1: 병원 찾기

1. 사용자가 지도 화면 진입
2. 현재 위치 기반 지도 표시
3. “동물병원” 선택
4. “24시” 필터 선택
5. 지도에 해당 병원만 표시
6. 마커 클릭 → 상세 정보 확인

---

### 시나리오 2: 반려동물 동반 카페 찾기

1. 지도 진입
2. “카페” 선택
3. 동반 가능 카페만 지도에 표시
4. 추천 카페 상단 노출
5. 마커 클릭 → 상세 정보 확인 → 전화걸기 또는 네이버 지도앱으로 길찾기
